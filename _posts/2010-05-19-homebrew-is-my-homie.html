--- layout: post title: Homebrew is my Homie tags: - General status: publish type: post published: true meta: _edit_last: '1' _wp_old_slug: '' --- If you're doing any kind of development on a Mac and you aren't using Homebrew, you're missing out. Homebrew is the best package manager for OS X by leaps and bounds. It makes it incredibly easy to install things like git, mysql, mongodb, node.js and tons more. All of the "Formula" are written in Ruby and hosted on GitHub, so it's easy to understand what's going on and to contribute. You can read more about it on the Homebrew home page or the Engine Yard blog, so I won't bore you with a long description here. Use it; you won't regret it.

--- layout: post title: Finding out the current version of stuff on Heroku tags: - General status: publish type: post published: true meta: _edit_last: '1' --- You can use the Herkou console to run system commands to get version info. For example, to get the current version of Bundler: 
heroku console %x{bundle -v}
...or the current version of ImageMagick: 
heroku console %x{identify -version}

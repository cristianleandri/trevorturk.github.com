--- layout: post title: Tantek tags: - General status: publish type: post published: true meta: {} --- The Box Model Hack is a famous trick that fixes the broken 'box' model in Internet Explorer 5 for the PC (the Mac version worked fine). The idea is simple, but complicated: basically, there are "user agents which misinterpret the CSS1 box model by placing border and padding inside the specified width". The problem is that the CSS box model is complicated, and some programmers got it wrong in IE.

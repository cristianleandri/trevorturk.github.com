--- layout: post title: El Dorado on Heroku tags: - General status: publish type: post published: true meta: _edit_last: '1' --- I'm pleased to announce that El Dorado is now compatible with Heroku, the instant Ruby platform. This means that deploying El Dorado is no longer a pain in the ass. It also means that you can get started with El Dorado for free. Just follow along with the installation and deployment instructions in the README and you're good to go. If you need help along the way, drop by the support site: http://eldorado.heroku.com/ Enjoy!
